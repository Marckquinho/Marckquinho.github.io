Overview

Thank you for purchasing this theme!

Ultra is fully Responsive Multipurpose Html5, CSS3, Bootstrap 3.x template. It's fully responsive design and retina ready, looks stunning on any device. It comes with a large variety of helpful features and page layouts for business, corporate, portfolio, products and marketing. It is easy to customize Ultra since it's built with developer-friendly and clean-coded in mind. We do not make any changes in bootstrap files, so can easily use bootstrap update & features
